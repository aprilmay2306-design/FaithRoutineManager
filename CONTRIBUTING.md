# Contributing Guidelines

Thank you for considering contributing.

This project values **ethics, calm design, and user autonomy** above feature growth.

---

## Core Principles (Non-Negotiable)

Contributions **must not** add:
- Analytics or tracking
- Internet dependency
- Ads or monetization
- Streaks, scores, or gamification
- Behavioral nudging or persuasion logic

---

## What Is Welcome

- UI polish and accessibility improvements
- Performance optimizations
- Bug fixes
- Documentation improvements
- Internationalization (translations)
- Code cleanup and refactoring

---

## Code Style

- Kotlin
- Jetpack Compose
- MVVM
- Clear naming
- Minimal dependencies

---

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make focused commits
4. Open a Pull Request with explanation

---

## Respect

This is a faith-aligned, calm technology project.

Please contribute with respect and humility.
